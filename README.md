# VoteCoin GUI wallet

Warning: this is development tree of VoteCoin GUI wallet for Windows.
Things are expected to be broken. Use at your own risk.
Do not use master branch under any circumstances. Use releases instead.

# How to start:

- Download a release
- Extract to a folder
- Run votecoin.exe
